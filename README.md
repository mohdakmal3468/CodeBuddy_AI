# 🤖 AI Code Reviewer

An AI-powered code review web app built for hackathon finals.

## 🚀 Features
- Bug detection & improvements
- Fixed code suggestions
- Beginner-friendly explanations
- Dark/Light mode
- Copy output
- Speak output (Text-to-Speech)

## 🛠 Tech Stack
- Flask (Python)
- HTML, CSS, JavaScript
- Google Gemini API

## ⚙️ How to Run Locally

```bash
pip install -r requirements.txt
